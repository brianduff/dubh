WELCOME TO NEWSAGENT

NewsAgent is a platform independent USENET Newsreader written in Java using
the Swing 1.0.2 user interface toolkit. This readme file contains late breaking
information that is not included in the documentation.

Please look at the 
   userdoc/about/whatsnew.html
documentation file for information on changes in this release of NewsAgent, or
use the "What's New?" menu item in the Help Menu inside NewsAgent.

Detailed information on NewsAgent is available on the World Wide Web at:
   http://st-and.compsoc.org.uk/~briand/newsagent/
   
The author can be contacted by emailing
   bduff@uk.oracle.com
   
(please note the change of email address from previous distributions. FROM

PLATFORM INDEPENDENT INFORMATION

o Configuration Files from Previous Versions

There have been a few changes to serialized preference files in this version
of NewsAgent. If you have configuration files from the previous version, you
will be informed how to remove them when NewsAgent starts up. 

This version of NewsAgent (1.0.2 build 110) is an alpha release for testing,
and should not be considered feature complete. Please check the web site
http://st-and.compsoc.org.uk/~briand/newsagent/download/alpha/
for updates.

To check which build of NewsAgent (and its dependencies) you are using, use
Help->About in the application.

WINDOWS SPECIFIC INFORMATION

The distribution now consists of a proper (InstallShield) installation
program and a binary stub file that executes the Java Runtime you
have installed on your computer with the correct CLASSPATH and
arguments. This should make NewsAgent more reliable and easier to
install.
