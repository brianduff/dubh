Customising the Toolbar

For information on customising the toolbar, see the Internationalising NewsAgent topic in Advanced Topics. The resource key for the toolbar is mainToolBar in Menus.properties. This resource can contain any number of menu commands. You can also define toolbar images for commands that don't have images: look at existing commands for examples. Images should be placed in the CLASSPATH (naclass directory on windows platforms).
