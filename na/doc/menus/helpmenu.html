The Help Menu

Help Menu Contents

The Help Menu provides access to the help system of NewsAgent and information about the author.

Help: Help Contents...

Keyboard Shortcut: F1
Toolbar Button:

Displays the root topic of the help system.

Help Topics

The various help topics in the Help menu take to to specific topics inside the help system hierarchy.

Help: About NewsAgent...

Keyboard Shortcut: None
Toolbar Button:  

Displays information about the author of NewsAgent and the current version number.
