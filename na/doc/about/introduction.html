Welcome to NewsAgent!

What is NewsAgent?

NewsAgent is an application which allows you to read USENET News. It was designed to solve many of the problems in current Newsreader software by trying to make sure you can find relevant articles more quickly.

NewsAgent is designed to be as easy as possible to use, but also contains many features that will be useful to the experienced user.
