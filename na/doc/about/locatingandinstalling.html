Locating and Installing NewsAgent

NEW: This topic has been significantly altered since version 1.02 of NewsAgent.

Locating NewsAgent

NewsAgent is freely available online. Because it's written in Java (a platform independent programming language developed by Sun Microsystems), it is available for all platforms which are capable of running the Java 1.1 virtual machine.

The main NewsAgent site is at:

http://st-and.compsoc.org.uk/~briand/newsagent/

Downloads and updates are available from thedownloads section on those pages.

A developer's source code release is available from DevCorner, the development section of the NewsAgent web site.

NewsAgent System Requirements

NewsAgent requires a platform which can run Java 1.1.5 (or later) applications. NewsAgent can run under some earlier versions of Java, but there are some problems. You cannot use NewsAgent with Java 1.0. JavaSoft provide a Java Runtime Environment for Microsoft Windows and Solaris UNIX platforms. Runtime Environments are available for many other platforms.

YOU MUST HAVE JAVA INSTALLED TO USE NEWSAGENT

Installing NewsAgent on Microsoft Windows

This section explains how to install NewsAgent on Microsoft Windows(TM). Follow these instructions:

The instructions for installing under Windows were incorrect in the user documentation included with versions 1.0 and 1.01 of NewsAgent. In version 1.02, a proper installation program has been included, and the instructions below have been modified so that they are up to date.

 

  • Go to the downloads section on the NewsAgent Web Site. Use the download wizard to determine which download is correct for you. This will depend on whether you already have the Swing toolkit installed on your computer.
  • Unzip the ZIP file into a temporary directory. You will need an application such as WinZIP to do this. PKUNZIP may not work correctly, since the ZIP file contains long filenames.
  • Run the setup.exe program. This program will check to see whether you have up to date versions of the JDK or JRE installed. You must have the JDK or JRE installed to proceed with setup. If you have downloaded the distribution of NewsAgent which doesn't contain Swing, you must tell the setup program where the swingall.jar file is located on your computer.
  • The setup program will prompt you for an installation directory. Choose any directory you like.
  • NewsAgent is now installed on your computer. You can delete the distribution ZIP file and any temporary files. If you ever want to uninstall NewsAgent, use the Add/Remove Programs... Control Panel. You can run NewsAgent from the Start Menu or by double clicking newsagent.exe in the installation directory.

Installing NewsAgent on Solaris (UNIX)

This section explains how to install NewsAgent on Solaris UNIX platforms. Follow these instructions:

  • Go to the downloads section on the NewsAgent Web Site. Use the download wizard to determine which download is correct for you. This will depend on whether you already have the Swing toolkit installed on your computer. The distribution will be contained in a tarred and gzipped archive file.
  • Change to the directory in which you want to install newsagent (e.g. /local/packages/) and unpack the archive file with the commands:
    gunzip naunix-102.tgz
    tar -xvf naunix-102.tar
  • NewsAgent is now installed on your computer. You may want to add the directory /local/packages/newsagent to your PATH environment variable to make it easier to run NewsAgent. To do this, edit your .tcshrc or .cshrc file and add the following line to the end:
    setenv PATH $PATH:/local/packages/newsagent/bin
  • You may now delete the tar file (newsagent-solaris.tar).

Installing NewsAgent on Other Platforms

NewsAgent will work on any platform which has a Java Runtime environment version 1.1.5 or later. More information on third party Java ports is available from JavaSoft.

  • Download and install the JRE or JDK 1.1.5 (or later) on your computer.
  • Use the download wizard on the NewsAgent web page, telling it about your system. If your system is not Windows 32 bit or Macintosh, you will be able to download the crossplatform version. It will be compressed using ZIP.
  • Unarchive the crossplatfrom distribution archive into a common directory (for instance /local/packages/newsagent on UNIX systems).
  • Add the following items to your CLASSPATH environment variable or equivalent. Replace /local/packages with the directory your installed newsagent into.
    • /local/packages/newsagent/swingall.jar
    • /local/packages/newsagent/newsagent.jar
  • Run NewsAgent using the command
    jre -cp %CLASSPATH% uk.ac.st_and.dcs.newsagent.NewsAgent
If you have already installed the Java Foundation Classes 1.1 (Swing), and the swingall.jar file is already in your CLASSPATH, you can delete the swingall.jar file from the newsagent directory to save space.
