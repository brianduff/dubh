Using the NewsAgent Help System

Help Files

The help files are distributed with NewsAgent, and can be found in the userdoc subdirectory of the directory in which NewsAgent was installed. To view these files, we recommend you use either Microsoft Internet Explorer 3.0 or later or Netscape Navigator 4.0 or later.

QuickHints

Some help pages contain QuickHints.

A quickhint is displayed in a green box with an exclamation mark icon to the left.

Quick hints may provide additional or summary information on the current topic, or hints on how to perform certain common tasks.

Help on the World Wide Web

From time to time it's necessary to update the NewsAgent help files because of mistakes or inaccuracies. The online version of the help files will always be up to date. You can either view them online, or download them and replace your existing help files.

The URL for the online help files is:

http://st-and.compsoc.org.uk/~briand/newsagent/userdoc/

You can download the help files in a convenient ZIP file from:

http://st-and.compsoc.org.uk/~briand/newsagent/downloads.html

The userdoc.zip file will uncompress into a directory called userdoc, which you should replace the old userdoc directory with.

Help Inside NewsAgent

You can access the help system from inside NewsAgent by invoking the Help: Help Contents menu command. This will open a browser to the contents page of the NewsAgent help system.

You can configure NewsAgent to use your favourite browser to display its help files. [Link]

 

The Help Menu has a number of other items that jump directly to important sections of the user guide.

Note that the internal web browser has a number of shortcomings. In particular, the icons for navigation don't work very well.
