Reading News

Navigating

In order to read news, you will have to be familiar with the user interface of NewsAgent's main window. The window contains three panels. You can adjust the size of these panels at any time. The three panels are shown in the diagram below.

Folders & Servers Tree Panel

A tree containing two top level nodes ("Servers" and "Folders"). Servers contains an icon for each News Server you are using. This icon changes depending on whether you are currently connected to the server or not. The server node contains icons for each newsgroup you are subscribed to on that server.

The Folders sub tree contains icons for all the folders you can save and retrieve messages to and from.
Message Header Display Panel

In this tree, each node is a message in the currently selected Newsgroup. Information about the message, such as who it was written by, the date it was posted and the subject are displayed in columns. You can adjust the width of these columns by dragging the separators between the column headers.

Clicking on any message displays it in the Message Display Panel.

Followups to a message are displayed as children in the tree. You can expand a message to see its followups by clicking on the icon to the far left of any header.
Message Display Panel

This panel displays the body text of the message which is currently selected in the Message Header Display Panel.

 

Viewing the List of Messages

To display the list of messages in a Newsgroup:

  • Locate the server on which the newsgroup is located in the Folders & Servers Tree. Expand it if necessary.
  • Select the newsgroup you wish to view (by clicking on it).
  • If you are not currently connected to the server, you will be asked if you wish to connect. You should answer "Yes".
  • NewsAgent will begin downloading message headers from the newsgroup. Progress is indicated by the progress meter on the right of the status bar.

Viewing a Specific Message

  • Using the instructions above, display the message list for the newsgroup you wish to view.
  • Click on the header of the messsage you want to view. The message will appear in the Message Display Panel after a short pause.
