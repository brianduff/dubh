Add / Edit Relevance Rule Wizard Step 3

Setting Action Conditions

The third step of the wizard allows you to define the action conditions that will change the display of headers depending on their score.

Condition List

This list contains plain english descriptions of all your scoring conditions.

Add / Edit

Adds or edits an action condition. The Action Condition Editor is displayed. Edit is only enabled if a condition is selected in the condition list.

Remove

Permanently removes the currently selected condition.

Move Up / Move Down

Change the order that scoring conditions are activated when a message header is being scored.
