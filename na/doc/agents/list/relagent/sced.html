The Scoring Condition Editor

Displaying the Scoring Condition Editor

Use the Relevance Rule Configuration Dialogue to add or edit a relevance rule. This will display the Relevance Rule Wizard. Click on the add or edit button in step 2 to bring up the scoring condition editor. For more information about scoring conditions, see the introduction.

Header Field

Enter the header field to compare. This is case insensitive, but should contain no spaces. If the header doesn't exist in the message, the score will not be adjusted.

Comparison Type

How to compare the header field to the text. Can be one of:

  • contains
  • starts with
  • ends with
  • is equal to

Comparison Text

The text to compare to the header.

Score Adjustment Value

A number to add to the score. This can be a negative number if you want to subtract a value. You should use integer values for scores. A real value will be truncated.
