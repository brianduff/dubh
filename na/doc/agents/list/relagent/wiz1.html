Add / Edit Relevance Rule Wizard Step 1

Describing Your Rule

After displaying an initial introduction screen, the first step of the relevance rule wizard lets you identify your rule and the newsgroups that it affects.

Rule Name

The name of your rule, as it will appear in the Relevance Rule Configuration Dialogue. You will be unable to change this if you are editing a rule.

Newsgroup Wildcard

A wildcard for newsgroups that this rule affects. For example * will match all newsgroups. comp.* will match only newsgroups that start with comp. (e.g. comp.lang.java.programmer, comp.sys.ibm.pc, but not alt.tv.simpsons).

This is a simple wildcard, and doesn't support advanced features such as single character wildcards. These may be added in future versions of the Relevance Agent.

Wizard Buttons

Use these buttons to navigate or cancel the wizard.
