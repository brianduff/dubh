Accessing the Dubh.Org CVS Repository

The easiest way to access the CVS Repository is to browse the tree online.

If you have a CVS client installed (see CVS Home), you can directly access the repository in read-only mode. To do this, set the CVSROOT environment variable to:

:pserver:guest@cvs.dubh.org:/usr/local/cvsroot

Now you can run cvs login to connect to the repository. When prompted for a password, enter guest.

Read / Write Access to CVS

Read / Write access will be given to anyone who wants to contribute to the projects being hosted on dubh.org. Please contact Brian (Brian.Duff@oracle.com) for more details.
