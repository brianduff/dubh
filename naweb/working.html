NewsAgent: SuperConnected!

Yet another tantalising screenshot from NewsAgent. I've been working over the last few weeks to tie the back & front ends of the application together. The back end involves the actual connections to the network and disk storage. Seems to be working quite well now, and for the first time, you can actually use NewsAgent to read news (about time, probably, since the project is due to be complete in a mere 2 weeks!)

Notes

Well, as you can see, a real, live news server connection is active, with a newsgroup selected and a specific message being shown. There are still a few missing features and undoubtedly some bugs, but it seems relatively stable, and not as slow as I had feared.
