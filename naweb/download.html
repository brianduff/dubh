The Mailing List

The NewsAgent mailing list is a moderated list; you will receive email about new versions of NewsAgent. Only the author and contributors can mail out to the list . Your email address will not be forwarded to third parties (other than the group hosting service ww.egroups.com). You can also subscribe / unsubscribe / view archived messages on the web at http://www.egroups.com/lists/newsagent-announce.

Development Code

You can now dowload code that is under development for NewsAgent 1.1.0. This code is automatically uploaded to the web every night if the source code has changed since the previous night. There are no guarantees that it works (or, at the moment, even compiles). This is intended to give contributors early access to source code. N.b. NewsAgent 1.1.0 requires Java 1.2.
