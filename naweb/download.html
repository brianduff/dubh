Next, fill in the form below. Information you enter is only stored if you choose to subscribe to one of the mailing lists, and is never passed on to third parties (except egroups.com for the mailing lists). Mandatory fields are in red.

The Mailing Lists

There are two NewsAgent mailing lists which you can subscribe to from this form. newsagent-announce is a low volume list that recieves email when announcements or new versions of NewsAgent become available. Only the author and contributors to NewsAgent can post to this list.

newsagent-discuss is an open email list which can be used for general discussions about NewsAgent, reporting bugs, making suggestions and asking for help. Any member of this list can send messages to it.

You can easily change your subscription to these lists from the egroups web site. See:

http://www.egroups.com/list/newsagent-announce

http://www.egroups.com/list/newsagent-discuss

Development Code

You can now dowload code that is under development for NewsAgent 1.1.0. This code is automatically uploaded to the web every night if the source code has changed since the previous night. There are no guarantees that it works (or, at the moment, even compiles). This is intended to give contributors early access to source code. N.b. NewsAgent 1.1.0 requires Java 1.2.
