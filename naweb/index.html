What is NewsAgent?

NewsAgent is a completely free 100% Java 1.1 USENET Newsreader. It was written by Brian Duff using the Java Foundation Classes (Swing).

As well as being a fully featured News reader, the program also has support for Agents: plugin modules which can affect the way messages are processed when reading or posting news articles. See the user documentation for details on the features it provides.
