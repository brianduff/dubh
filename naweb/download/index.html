NewsAgent version 1.0.2 (Build 110)

For Windows 95, 98, NT

The Windows installer includes an installshield installer for the program that creates convenient start menu shortcuts for launching NewsAgent. It includes NewsAgent 1.0.2 Build 110, Dubh Java Utilities 1.0.1 Build 10 and Swing 1.0.2. Full HTML user documentation is also included. It requires JDK or JRE 1.1.5 or greater to be installed on your machine. Java 2 is not currently supported.

  • Windows ReadMe
  • Windows zipped installer [2.85Mb] (includes Swing 1.0.2)

Platform Independent

Source Distribution

There are source code distributions for NewsAgent 1.0.2 and Dubh Java Utilites 1.0.1. There are source only distributions (containing just .java source files), and build distributions (containing a full build environment, including utility scripts and nmake compatible makefiles). The build distributions are for use under Windows NT only, and require you to have Microsoft nmake installed on your computer (I'll upgrade to GNU Make some time soon). More information is available in developers corner.

  • NewsAgent 1.0.2 build 110 source only [283 Kb]
  • NewsAgent 1.0.2 build 113 source and build tools [607 Kb]
  • Dubh Utils 1.0.1 build 5 source only [100 Kb]
  • Dubh Utils 1.0.1 build 5 build environment [469 Kb]
