NewsAgent 1.1.0 Release Schedule

Clearly the contents of this page are a pile of rotting fish remains. I really shouldn't make half arsed promises I'll never keep <sigh>.

These are approximate release dates for various milestones of NewsAgent 1.1.0. Being a somewhat lazy bugger occasionally, these almost certainly will slip (I have a slightly sceptical view of the concept of a deadline. This is a natural and required trait for any software engineer).

The feature set listed here assumes that no-one else offers to contribute. I'm beginning to get the impression that open source software has to be half decent before anyone will contribute to it, and NewsAgent isn't quite there yet. If, however, the universe stuns me completely, and some contributors do turn up, the release schedule could quicken, or find itself burdened with many more features. (for the record, there are a couple of registered contributors with CVS access to the source, but they don't seem to be very active).

The idea is that milestone releases will be available as installable alpha downloads of NewsAgent. These should be relatively stable builds. The features for each milestone are split into "musts" and "shoulds". A milestone will slip if a must isn't going to make it into the release. Shoulds can slip to become musts in later milestones.

So, I'm not promising anything, but here goes... (btw. the milestones are all named after characters from scifi novels. Yes, I am sad.)

Milestone Gurgeh: 30 November 1999 (pre-alpha) 

Musts

  • Functioning Javamail implementation of NNTP (DMP 1.0.0)

Shoulds

  • New Swing 1.1 header tabbed tree
  • All dialogs language independent & layout improved

Milestone Balveda: 31 January 2000 (alpha 1)

Musts

  • Message Composer with HTML and attachment support
  • Message Viewer with HTML and attachment support, fully configurable
  • POP3 / SMTP / IMAP email support
  • Improved newsgroup subscriptions dialog

Shoulds

  • Fully configurable main window (menus, toolbars, panes, etc.)

Milestone Zakalwe: 31 March 2000 (alpha 2)

Musts

  • Caching
  • Offline Support

Shoulds

  • Help doc / integration

Milestone Chmee: 15 May 2000 (beta 1)

Musts

  • ECMAScript support

Milestone Thermopyle: 30 June 2000 (release candidate)

Bug fix release / slippage space

Milestone Dios: 15 July 2000 (release 1.1.0)

Bug fix release / slippage space
