Planned Features for NewsAgent

In the next version (1.02)

  • A better designed 'Subscriptions' form, allowing users to view Newsgroups hierarchically and search for newsgroups containing specific words.
  • Improvements to the Agent interface: allowing agents to add menu or toolbar commands and commands in popup menus.
  • Signatures and email options properly implemented. Folder storage fixed.
  • Caching fixed properly.
  • Easier installation for Windows.

At some point after version 1.02

  • Mime type / HTML support.
  • Hyperlinked message IDs and http://... text.
  • A decent help system
  • Proper multithreaded access to NNTP server so that user can read messages while the header list is still being downloaded from the server.
  • Performance improvements?
  • Column sorting in the message header list display. Probably using the collection classes in the final version of JDK 1.2
  • Drag & drop, as soon as it is added to Java.

Anything else?
