Overview of the project

The FileSystem Browser project creates a top level node in the JDeveloper system navigator which allows you to browse the file system and work with files without adding them to a project.

For usage details, see Usage Info page.

The files of this project

This project consists of the following files along with their purpose.

File Name File Purpose
fsbrowser.jpr The JDeveloper Project file.
build.xml Ant buildfile for the project. Information on building with Ant is below.
ProjectInfo.html The document you're currently reading
FileSystemBrowser.java This file is the addin, which creates menus and registers the top level node in the navigator.
FileSystemElement.java

The top level file system node.
FileSystemFolderElement.java A DefaultFolder subclass that represents a folder on the file system.
JARNavigatorNode.java A DefaultFolder subclass that provides access to the contents of a JAR or ZIP file from inside the navigator.
manifest.mf The manifest to be associated with the delivered jar-file, that contains instructions for the auto-loader.

Building this project

You can build this project in the JDeveloper IDE using the Build Project command on fsbrowser.jpr.

To build this project using Ant, download and install Ant from http://jakarta.apache.org/ant. In the fsbrowser directory, type:

ant -Djdevhome=location_of_jdev all

Replace location_of_jdev with the path to a JDeveloper installation.
