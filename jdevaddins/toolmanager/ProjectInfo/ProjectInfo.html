Overview of the project

The purpose of this project is to provide a way to easily define menu and toolbar items in the IDE which invoke external programs.

For usage details, see Usage Info page.

The files of this project

This project consists of the following files along with their purpose.

File Name File Purpose
toolmanager.jpr The JDeveloper Project file.
build.xml Ant buildfile for the toolmanager project. Information on building with Ant is below.
ProjectInfo.html The document you're currently reading
ToolManager.java This file is the addin, which creates the Edit Tools... menu item. It is also responsible for loading and saving the tool definitions and creating menu and toolbar items for them at IDE startup.
ToolArgument.java

An interface that describes a type of argument that can be passed in to external programs when invoked from the Tools menu.

For more information on writing new argument types, see the Extending the ToolManager section below.
ToolPanel.java The user interface for the Edit Tools dialog
tool.gif The default toolbar icon for new tools.
argument package The files in the oracle.jdevimpl.toolmanager.argument package are implementations of ToolArgument which come with the addin. These can be used as examples for writing new argument types. See the Extending the ToolManager section below
manifest.mf The manifest to be associated with the delivered jar-file, that contains all the instruction for the auto-loader.

Extending the ToolManager

The ToolManager contains a number of predefined argument types in the argument package. These argument types appear in the Edit Tools dialog when you add an argument or select the dropdown beside the "Run in Directory" field.

The available argument types can easily be extended by implementing the ToolArgument interface and registering your argument type with the ToolArgumentRegistry. You can do this in the initialize() method of an addin, provided the addin is loaded after the ToolManager.

ToolArgument myArgument = new MyArgumentClass();
ToolManager.getArgumentRegistry().register( myArgument );

Building this project

To build this project in the JDeveloper IDE, simply open toolmanager.jpr and compile it using the Build menu item.

To build this project using Ant, download and install Ant from http://jakarta.apache.org/ant. In the toolmanager directory, type:

ant -Djdevhome=location_of_jdev all

Replace location_of_jdev with the path to a JDeveloper installation.
