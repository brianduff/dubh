ToolManager extension

This tool allows you to easily add menu and toolbar items to the JDeveloper to invoke external applications from within the IDE.

This tool is part of the JDeveloper extension exchange.

Additional Details

Dependencies: none
Limitations: accelerators defined in Preferences->Accelerators are not persisted properly.
Compatibility: Oracle9i JDeveloper version 9.0.2.7.55 or later [internal note: this addin will not work with earlier 9.0.2 builds, and is already part of the product in 9.0.3 builds]
Extension Version: 0.4
