TemplateMaker extension

This tool provides the ability to define and use templates in the IDE. The templates can use the powerful features of the Jakarta Velocity template engine (see http://jakarta.apache.org/velocity).

This tool is part of the JDeveloper extension exchange.

Additional Details

Dependencies: Jakarta Velocity 1.2 (included)
Limitations: When a template is first created, it can't be deleted until the IDE is restarted.
Compatibility: Oracle9i JDeveloper 9.0.2
Extension Version: 0.3
