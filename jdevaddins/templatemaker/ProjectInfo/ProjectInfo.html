Overview of the project

The TemplateMaker project provides support for defining and using templates in the IDE using the Jakarta Velocity template engine.

For usage details, see Usage Info page.

The files of this project

This project consists of the following files along with their purpose.

File Name File Purpose
templatemaker.jpr The JDeveloper Project file.
build.xml Ant buildfile for the templatemaker project. Information on building with Ant is below.
ProjectInfo.html The document you're currently reading
TemplateMaker.java This file is the addin, which registers the "Save as Template..." menu item, programmatically adds templates to the object gallery and creates the Templates folder in the IDE system navigator.
TemplateCaster.java

Interface that is used when TemplateMaker creates an output file from a template. There are two implementations of this interface. CopyTemplateCaster simply copies the template to the output path whereas VelocityTemplateCaster uses the Jakarta Velocity template engine to provide powerful template scripting capabilities.
TemplatesFolder.java The navigator node that provides convenient access to template definitions from inside the IDE. It's really just a view onto the templates subdirectory of JDEV_USER_HOME. Each item under this folder is an instance of TemplateNode.
UseTemplateDialog.java The user interface component used in a dialog when the user uses the object gallery to create a new file based on a template. This dialog is invoked from the TemplateWizard.
TemplateWizard.java "wizard" used by the object gallery to invoke the UseTemplateDialog. A lot of code in this class is centered around defining sensible default paths or packages for the new file based on an IDE context. This class also invokes the TemplateCaster to actually transform a template into an output document.
IDELogSystem.java Binds the log output from Jakarta Velocity to the IDE log window
URLResourceLoader.java Provides a way for Jakarta Velocity to load templates specified by URL using the IDE's URLFileSystem.
URLWrapper.java Wrapper round a URL that provides convenient access to information about URL components to a template writer. Used by the VelocityContextObject.
VelocityContextObject.java This object serves as the class for the oracleide resource which is available to template writers when writing Velocity templates.
VelocityTemplateCaster.java A TemplateCaster implementation that uses the Jakarta Velocity template engine to generate output documents from templates.
velocity-dep-1.2.jar The Jakarta Velocity template engine. See http://jakarta.apache.org/velocity for more details on Velocity.
velocity.LICENSE.txt The license for Jakarta Velocity.
manifest.mf The manifest to be associated with the delivered jar-file, that contains instructions for the auto-loader.

Building this project

To build this project in the JDeveloper IDE, you must define the Jakarta Velocity library.

  1. In Project Settings... in the Project menu, go to the Libraries page.
  2. Select the Apache Velocity library (it will be highlighted in red)
  3. Click Edit... set the Class path to include the velocity-dep-1.2.jar which is included in the distribution.
  4. Click OK to save the project settings.

You should now be able to compile the project.

To build this project using Ant, download and install Ant from http://jakarta.apache.org/ant. In the templatemaker directory, type:

ant -Djdevhome=location_of_jdev all

Replace location_of_jdev with the path to a JDeveloper installation.
