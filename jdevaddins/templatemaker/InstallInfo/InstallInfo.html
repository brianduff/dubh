TemplateMaker Installation Instructions

This tool is packaged as a JDeveloper autoload extension.

[back to home page]

To Install this extension:

Download Jar file to $JD9i/lib/ext

Download templatemakerjar.zip and extract the two embedded Jar files into the JDeveloper path $(JD9i)/lib/ext.

Re-Start JDeveloper

  • Exit JDeveloper if already running
  • Restart JDeveloper and JDeveloper will automatically load all autoload Jar files

For more info, please see JDeveloper's Online help
