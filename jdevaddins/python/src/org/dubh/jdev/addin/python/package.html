Contains an addin and classes to support Jython in the JDeveloper IDE

Features Installed by this Addin

The PythonAddin class is a JDeveloper addin which adds support to the IDE for the Jython language. For more information on Jython, visit http://www.jython.org.

Following are the features this addin installs into JDeveloper:

Python Node Type

PythonNode represents a custom node type in the IDE for .py files.

Python Category

PythonCategory is a category folder for Python scripts in the IDE navigator.

Editor Language Module for Python

The addin installs a language module into JDeveloper's code editor which supports syntax highlighting of .py files. More information on the language module is available in the org.dubh.jdev.language.python package.

Interpret Python Script Menu Item

The addin installs a menu item in the context menu of .py nodes to allow the Jython interpreter to interpret a script. The full IDE environment is available to any python script run through the menu or the console (see below).

Python Console

The python console is an interactive console window inside the IDE which can be used to run and inspect the output of python commands. The IDE can be controlled through its public Java APIs from within the python console.

Python Category in Object Gallery

Provides the ability to create new python scripts from the IDE object gallery.
