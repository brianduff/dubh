Python Addin Installation Instructions

This tool is packaged as a JDeveloper autoload extension.

[back to home page]

To Install this extension:

Download Jython

This addin requires Jython, the Java Python interpreter. To install Jython:

  • Download and extract Jython from http://www.jython.org. The addin was developed against Jython 2.1.
  • Copy jython.jar and the Lib directory from under your Jython installation to the JDeveloper path $(JD9i)/lib/ext

Download JDeveloper Python Support Addin Jar file to $JD9i/lib/ext

  • Download jdpythonjars.zip and extract the two enclosed JAR files (jdpython.jar and dubhconsole.jar) into the JDeveloper path $(JD9i)/lib/ext.

Re-Start JDeveloper

  • Exit JDeveloper if already running
  • Restart JDeveloper and JDeveloper will automatically load all autoload Jar files
