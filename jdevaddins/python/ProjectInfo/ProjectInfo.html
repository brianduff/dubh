Overview of the project

The Python Addin for JDeveloper brings support into the IDE for the Python Language.

For usage details, see Usage Info page.

The files of this project

This project consists of the following files along with their purpose.

File Name File Purpose
python.jpr The JDeveloper Project file.
build.xml Ant buildfile for the project. Information on building with Ant is below.
ProjectInfo.html The document you're currently reading
org.dubh.jdev.addin.python package This package contains code to install the menu items, dockable windows, gallery items and preference pages into the IDE which make up the Python addin.
org.dubh.jdev.parser.python package

This package contains a lexer for Python syntax. A lexer is used to determine where the tokens are in a source file. The implementation of this lexer is based on the Python 2.2 Language Reference Manual.
org.dubh.jdev.language.python This package contains language support for JDeveloper's code editor to support the Python language. This includes a description of the styles used by highlighted tokens and classes for rendering Python code.
org.dubh.ui.console (in the console.jpr project) The Console component is a separate general purpose UI utility which extends JTextPane to provide console-like behavior, including constrained input, a command history and color / font styles.
manifest.mf The manifest to be associated with the delivered jar-file, that contains instructions for the auto-loader.

Building this project

To build this project, you must download Jython as described in the Installation Guide. Copy jython.jar into the jdpython directory from the expanded project zip.

You can build this project in the JDeveloper IDE using the Build Project command on python.jpr. You will need to define the Jython Library. On the Libraries page of Project Settings, you will see the Jython library highlighted in red. Select Jython and click the Edit... button. Set the classpath to the location of the downloaded jython.jar file. In addition, you will need to build console.jpr from the console directory, and make sure the compiled classes are available to the jdpython project.

To build this project using Ant, download and install Ant from http://jakarta.apache.org/ant. Make sure you have copied jython.jar to the jdpython directory, then type:

ant -Djdevhome=location_of_jdev all

Replace location_of_jdev with the path to a JDeveloper installation.
